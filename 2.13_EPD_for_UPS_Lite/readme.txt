Instructions for using this board on the Pwnagotchi 

The pin definition on the e-ink screen circuit of this board is compatible with the waveshare  2.13 inch e-Paper HAT V4, 
which means that when you configure the Pwnagotchi screen, the screen type can be selected "waveshare_4"
ui.display.type = "waveshare_4"

For the use of the RTC chip DS1307Z, please refer to the detailed tutorial on Adafruit website
https://learn.adafruit.com/adding-a-real-time-clock-to-raspberry-pi

For the case, you can download the case 3D file, print and install it yourself
https://github.com/linshuqin329/UPS-Lite/tree/master/UPS-Lite_Case/UPS-Lite_EPD_Case
https://github.com/linshuqin329/UPS-Lite/tree/master/UPS-Lite_Case/UPS-Lite_EPD_Antenna_Case

关于这个板子在Pwnagotchi 项目上的使用说明，

这个板子的墨水屏电路上的引脚定义与微雪的2.13寸墨水屏V4版本是兼容的，也就是说当你在配置Pwnagotchi 屏幕的时候，屏幕类型可以选择“waveshare_4”即可
ui.display.type = "waveshare_4"

关于RTC芯片DS1307Z的使用，请参考Adafruit网站上的相关详细教程
https://learn.adafruit.com/adding-a-real-time-clock-to-raspberry-pi

关于配套的外壳，可下载外壳3D文件，自行打印安装
